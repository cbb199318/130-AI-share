# 30 套 UI 风格英文 Prompt

> 使用方式：
> 将以下任意一条 **UI Style Prompt** 直接追加到你的 **App 首页结构 Prompt** 末尾即可。
>
> 推荐格式：
> ```
> [App Home Structure Prompt]
>
> Apply the following UI style:
> [UI Style Prompt]
> ```

---

## 01. Minimal iOS Style（极简 iOS 风格）
```text
Apply a minimal iOS-style interface.
Use generous white space, soft shadows, rounded cards, and subtle dividers.
Typography should feel clean and system-native.
```

## 02. Apple Fitness Style（Apple 健身风格）
