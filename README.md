## 仓库结构

```
AI-share/
├── prompts/         # 各类提示词（Prompt）集合
│   ├── figma/       # Figma Make 相关 Prompt
│   ├── chatgpt/     # ChatGPT 相关 Prompt
│   └── ...
├── tutorials/       # 教程与使用指南
├── examples/        # 示例项目或结果展示
└── README.md        # 仓库说明
```

## 使用指南

1. 浏览 `prompts/` 文件夹，根据分类选择需要的 Prompt。
2. 阅读对应教程或示例以了解使用方法。
3. 可以直接复制 Prompt 到对应 AI 工具中进行使用。
